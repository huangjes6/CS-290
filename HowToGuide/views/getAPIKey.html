Get A Steam Web API Key

Before we can start using the Steam Web API to do some cool things, we need to get an API key. Keys are unique codes that are used to identify the user. Think of it as a social security number. It's not something you want to show other people or allow others to use. To get a key, go to the Steam Web API Documentation page, go to the "Obtaining an Steam Web API Key" section, and follow the instructions there. If you don't already have a steam account, create one now. (Note that you need to have made at least a $5 purchase on Steam before you are allowed to get an API key.) If you already have an account, log in and you will be directed to this page.

The domain page field is not very important and is not actually used, so you can just enter something that will help you remember what your website or app is. After clicking "Register", you will receive your Steam Web API Key and be given an option to revoke the key, which you don't want to do because we need that key! Now that we have the key, we can use it to request information from Steam. Keep in mind that we are limited to 100,000 calls per day to prevent call spamming. Let's get started!
