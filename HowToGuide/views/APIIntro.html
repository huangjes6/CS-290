What's an API?

Application Program Interface, or API for short, are libraries which can be used to build software applications. They allow software components to communicate with each other in order to get data. You've probably already used many websites and apps that were created with APIs. Check out this video for a short explanation of APIs.

The Steam Web API

The Steam Web API is a HTML based Web API. It has both public and private methods. The public methods can be accessed through the use of HTML requests and these are the ones we will focus on. Through the use of these methods we can get news on a specific game, get the achievements possible for that game, and much more. The default response format is JSON, but each method can return results in XML and VDF as well if specified. For this tutorial, we will stick with JSON (JavaScript Object Notation), which is an easy to read and easy to parse data-interchange format. Before we start, we need an Steam API Key.
